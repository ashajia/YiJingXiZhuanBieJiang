# 《易经系传别讲》
## 作者：南怀瑾
> 《易经系传别讲》是南怀瑾先生对《易经》的精辟讲述，也是南怀瑾先生对孔孟思想、儒家学说的探源。从自然的道理，说到人文的精神，人生的道理，修行的道理。无论入世出世，均为不可亟得的摩尼宝典。读了《易经系传别讲》，你会发现《易经》是很平易的一种学问，是人人都需要的一种学问。最重要的是，本书会使你豁然开朗，智慧大增。
